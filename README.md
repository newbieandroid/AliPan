# 阿里云盘列表程序 - Spring Boot

## 演示及开发环境

> https://pan.rawchen.com
>
> JDK 8 + IDEA 2021.2.3

## 使用（暂时）

* Windows或Linux下载安装 JRE 8 或 JDK 8 及以上版本，下载ZIP解压为项目文件夹。IDEA打开，配置yml中的refresh_token(刷新令牌)和parent_file_id(根文件夹)，Maven打包jar运行。
* 以后会添加jar包直接运行，后台配置参数。
